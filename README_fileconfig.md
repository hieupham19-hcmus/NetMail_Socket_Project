
# Config File Formats

Config fix to use our application
## TXT Format
```txt
Username: value
Password: value
Email: value
MailServer: value
SMTP: value
POP3: value
Autoload: value
```

## XML Format
```xml
<config>
    <Username>value</Username>
    <Password>value</Password>
    <Email>value</Email>
    <MailServer>value</MailServer>
    <SMTP>value</SMTP>
    <POP3>value</POP3>
    <Autoload>value</Autoload>
</config>
```
## CFG Format
```cfg
Username = value
Password = value
Email = value
MailServer = value
SMTP = value
POP3 = value
Autoload = value
```

## JSON Format

```json
{
    "Username": "value",
    "Password": "value",
    "Email": "value",
    "MailServer": "value",
    "SMTP": "value",
    "POP3": "value",
    "Autoload": "value"
}
```

## INI Format

```ini
[Settings]
Username = value
Password = value
Email = value
MailServer = value
SMTP = value
POP3 = value
Autoload = value
```
