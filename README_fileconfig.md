
# Config File Formats

Config fix to use our application
## TXT Format
```txt
Username: value
Password: value
Email: value
MailServer: value
SMTP: value
POP3: value
AutoLoad: value
```

## XML Format
```xml
<config>
    <Username>value</Username>
    <Password>value</Password>
    <Email>value</Email>
    <MailServer>value</MailServer>
    <SMTP>value</SMTP>
    <POP3>value</POP3>
    <AutoLoad>value</AutoLoad>
</config>
```

## JSON Format

```json
{
    "Username": "value",
    "Password": "value",
    "Email": "value",
    "MailServer": "value",
    "SMTP": "value",
    "POP3": "value",
    "AutoLoad": "value"
}
```
